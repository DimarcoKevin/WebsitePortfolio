KevFlix

  • Kevflix ( Download)

    • DISCLAIMER: This download link does not contain any shows/movies or any other copyright materials. This project is a simple interface to display your own media collection with some features to make it more user friendly.

    Kevflix is a personal project I've been working on since 2023 with a subtle nod to a streaming site of a very similar name. It's original task was simply to give a clean front-end for my own movie and tv show collection. It slowly evolved with more features including search, sorting, and filtering by genre.

    In the latest beta release of Kevflix, I've added the ability to connect to an API that will get details and images for each of your movies and tv shows. It can also download the descriptions of every episode for a tv show and display them in the tv popup.

    This project is currently only available for Windows OS. There are still plans to advance this project even more and possibly make it open source so others can contribute to it! If you wish to help contribute to Kevflix or have any issues/questions feel free to contact me at kevflixapp@gmail.com
